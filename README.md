# Sergio Verga Portfolio

Personal portfolio site built with Next.js (App Router), TypeScript, and Tailwind.

![alt text](image.png)

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Lint, typecheck, format

```bash
npm run lint
npm run typecheck
npm run format
```

## Content updates

- Site metadata and bio: `content/site.ts`
- Projects list: `content/projects.ts`
- Experience: `content/experience.ts`
- Education: `content/education.ts`
- Writing: `content/writing.ts`

Static assets live in `public/` and `public/images/`.
